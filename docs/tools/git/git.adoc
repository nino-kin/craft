改行コードの扱いを制御


.gitattributesという名前のテキストファイルを作り、そこにtext属性に関する設定を書いてGitにコミットすれば、そのGitリポジトリを使う人全員で改行コードの扱いを統一できます。

```txt
*           text=auto
*.txt       text
*.vcproj    text eol=crlf
*.sh        text eol=lf
*.jpg       -text
```
